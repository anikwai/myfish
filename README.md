A modern Laravel application with Inertia.js and React for building interactive web experiences.

## Getting Started

```bash
composer install
npm install
cp .env.example .env
php artisan key:generate
php artisan migrate
```

## Development

```bash
composer dev
```

## Testing

```bash
./vendor/bin/pest
```
